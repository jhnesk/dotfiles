dotfiles
========

My dotfiles.

See <https://dotfiles.github.io/> for more.

Installation
------------

I have made a very simple shell script for installation, creating
symlinks.

1. Clone the repository.
	git clone git@github.com:jhnesk/dotfiles.git
2. Change directory.
	cd dotfiles
3. Run install script.
	./install.sh
